## Installation
> using python 3.11.2

### set python virtual environment
* Linux  
    ```sh
    $ cd "Machine Learning & Bioinformatics Lab"
    $ activate_venv.sh
    (.venv) $
    ```
* Windows  
    ```cmd
    > cd "Machine Learning & Bioinformatics Lab"
    > activate_venv.bat
    (.venv) >
    ```

### install packages
```shell
(venv) $ pip install -r requirements.txt
```
